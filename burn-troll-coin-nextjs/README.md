# 🔥 Burn Troll Coin

App para queimar o token Troll Coin na rede Solana Mainnet.

## 🚀 Como rodar

1. Instale as dependências:
```bash
npm install
```
2. Rode o app:
```bash
npm run dev
```

## 🌐 Acesse
- Conecte sua carteira Phantom ou Solflare.
- Clique em 🔥 **Burn Troll Coin**.
- Veja a transação no Solscan.
